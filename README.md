# ⚔️ TaskFlow // Sistema de Gestión del Gremio

TaskFlow es una herramienta de despliegue táctico diseñada para la gestión eficiente de misiones y encargos. Este sistema permite a los comandantes del Gremio organizar, filtrar y priorizar objetivos en tiempo real, optimizando la respuesta operativa ante amenazas de cualquier Rango.

---

## 🚀 Funcionalidades Principales (Protocolo Rango S)

El sistema ha sido ampliado mediante soporte de IA para incluir capacidades avanzadas de procesamiento:

* **🔍 Búsqueda Dinámica:** Localización instantánea de informes mediante palabras clave.
* **🎭 Filtros Multicapa:** Segregación de misiones por Estado (Logradas/Pendientes), Rango (S-D) y Categorías (Caza, Escolta, etc.).
* **✎ Edición de Campo:** Capacidad de rectificar detalles de misiones activas sin interrumpir el flujo de datos.
* **⚖️ Ordenación por Prioridad:** Algoritmo de jerarquización que posiciona automáticamente las misiones críticas (Rango S) en la vanguardia del tablón.
* **📊 Monitor de Progreso:** Visualización en tiempo real del rendimiento del Gremio mediante una barra de progreso dinámica.

---

## 🛠️ Tecnologías del Gremio

* **HTML5 & CSS3:** Estructura base con semántica profesional.
* **Tailwind CSS:** Diseño táctico responsivo con estética "Parchment & Gold".
* **Vanilla JavaScript (ES6+):** Motor lógico centrado en la manipulación eficiente del DOM y persistencia en `LocalStorage`.
* **Cursor AI & Gemini:** Asistencia táctica para la optimización de algoritmos complejos y depuración de errores.

## 📥 Instalación y Despliegue

1.  Clona este repositorio en tu terminal de mando:
    `git clone [https://github.com/OSCARaluceac/Tablon_De_Misiones.]`
2.  Accede al directorio del proyecto:
    `cd TaskFlow`
3.  Ejecuta el archivo `index.html` en tu navegador preferido. No se requieren dependencias externas adicionales gracias al uso de CDNs tácticos.
